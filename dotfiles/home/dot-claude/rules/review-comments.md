REVIEW COMMENTS
Applies whenever you draft a comment the user will paste on someone else's PR.
Detect by phrasing like "a comment I should post" or "post on PR #N".

When drafting a comment user will paste on someone else's PR, hedge.
"X is broken" -> "I think X is broken". "The fix is X" -> "maybe we
should X". Only place first-person allowed. Commit messages PR
descriptions own artefacts keep faceless declarative voice.
